# 🎨 Frontend - React Uygulaması

## 📋 İçerik
Bu klasörde Hesap Makinesi uygulamasının **frontend** kullanıcı arayüzü yer almaktadır.

## 🛠️ Teknolojiler
- **React 18** - UI Framework
- **JavaScript ES6+** - Modern JavaScript
- **CSS3** - Styling ve animasyonlar
- **React Hooks** - State yönetimi (useState)

## 📁 Dosya Yapısı
```
Frontend/
├── src/
│   ├── bilesenler/
│   │   ├── HesapMakinesi.js      # Ana hesap makinesi component'i
│   │   └── GecmisListesi.js      # İşlem geçmişi component'i
│   ├── servisler/
│   │   └── HesapMakinesiApi.js   # Backend API çağrıları
│   ├── App.js                    # Ana uygulama component'i
│   ├── App.css                   # Ana stil dosyası
│   ├── index.js                  # React giriş noktası
│   └── index.css                 # Global stiller
├── public/
│   ├── index.html               # HTML template
│   └── favicon.ico              # Site ikonu
├── package.json                 # NPM bağımlılıkları
└── README.md                    # Detaylı dokümantasyon
```

## 🚀 Çalıştırma
```bash
npm install
npm start
```

## 🎯 Özellikler
- **5 sütunlu grid** tasarım
- **Renkli buton** sistemi
- **Türkçe sayı formatı** (virgül desteği)
- **Real-time** hesaplama
- **İşlem geçmişi** görüntüleme
- **Responsive** tasarım

## 🔗 Port
Frontend varsayılan olarak **localhost:3000** portunda çalışır.
