# HesapMakinesiAPI
ASP.NET Core ile geliştirilen hesap makinesi API'si
Bu projede, dört temel matematiksel işlemi (toplama, çıkarma, çarpma, bölme) yapabilen bir ASP.NET Core Web API tabanlı hesap makinesi uygulaması geliştirdim. Uygulama, negatif ve pozitif sayılar ile birlikte ondalıklı sayılarla işlem yapma yeteneğine sahiptir.

Özellikler
🧮 Dört temel işlem desteği

➕➖ Negatif ve pozitif sayılarla işlem yapılabilir

🔢 Ondalıklı sayılar (double) desteklenmektedir

🧠 İşlem geçmişi özelliği ile yapılan son 10 işlem görüntülenebilir

💾 SQL veritabanı bağlantısı sayesinde işlem geçmişi kalıcı olarak kaydedilir

🖥️ Frontend kısmı Visual Studio Code ortamında hazırlandı

🧩 Backend kısmı ASP.NET Core kullanılarak Visual Studio ortamında geliştirildi



Teknolojiler
ASP.NET Core Web API

Entity Framework Core

SQL Server (Veritabanı bağlantısı için)

HTML / CSS / JavaScript (Frontend arayüz)

Visual Studio & Visual Studio Code
